location
(optional)
43211;43224
String
Location details (address, county, neighborhood or Zip code). It is required if the polygon or coordinates is empty.

The maximum number of locations (city, zip code) is 5, separate location by ;

Multi location /propertyExtendedSearch?location=78013;78014;New York, NY

The maximum number of locations is 5

Max length: 100

Please note that for the ZIP Code Type PO Box there will be empty results.
